# 디자인 가이드 (모바일 읽기 경험)

## 디자인 방향
- 따뜻하고 차분한 분위기(취침 전 독서에 적합)
- 시선 분산 최소화, 텍스트 중심 구성
- 카드/섹션 구분은 명확하지만 과한 장식은 지양

## 타이포그래피
- 본문: 고전적이고 읽기 쉬운 세리프 계열
  - 후보: "Noto Serif KR", "Gowun Batang"
- 제목: 산뜻한 산세리프 계열
  - 후보: "Pretendard", "Noto Sans KR"

## 색상 방향 (제안)
- 베이스: 크림/아이보리 계열 배경
- 포인트: 따뜻한 오렌지/브라운 계열
- 대비: 텍스트는 충분한 명도 대비 확보

## 읽기 화면 UX
- 큰 기본 글씨(모바일 기준 18~20px)
- 줄 간격 여유(1.6~1.8)
- 페이지 로드 시 초점은 본문에

## 컴포넌트 원칙
- 필터는 한 줄 스크롤 가능한 칩 형태
- 버전 전환은 명확한 라벨(연령/길이) 표시
