# MVP 콘텐츠 샘플

## 작성 기준
- 연령: 3-5 / 6-7 / 8-9
- 길이: 짧음(3~5분, 공백 제외 700~1100자) / 보통(6~10분, 공백 제외 1100~1700자) / 김(11~20분, 공백 제외 1700~2500자) / 짧은 연작(3~4화, 회차당 공백 제외 700~1700자) / 긴 연작(5화 이상, 실제 생성은 보통 5~8화)
- 태그: 공통 태그 목록에서 선택
- 연작은 chapters 배열로 회차 분리
