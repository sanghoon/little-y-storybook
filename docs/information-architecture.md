# IA / 주요 화면 구조

## 사이트맵 (초안)
- 홈
- 탐색/검색
- 태그별 리스트
- 읽기 화면(메타+본문 통합)
- 시리즈 페이지
- About/문의(선택)

## 핵심 사용자 흐름
1) 홈 진입 → 태그 필터 선택 → 리스트 탐색 → 읽기(메타/본문/회차)
2) 검색 → 결과 리스트 → 읽기 → 필요 시 버전 전환
3) 시리즈 진입 → 회차 선택/스크롤 → 이어보기

## 화면별 핵심 요소
### 홈
- 추천/신규/인기 섹션
- 빠른 필터(연령/길이)
- 이어보기(최근 N개)

### 리스트(탐색/태그)
- 카드: 제목, 연령, 길이, 테마
- 필터 바: 연령/길이/테마/분위기

### 통합 읽기
- 이야기 소개/요약/태그/버전 메타
- 관련 버전 바로가기
- 본문 읽기(단편/연작)
- 글자 크기 조절
- 진행 표시(스크롤 %)
- 이어보기(마지막 읽은 회차/위치 복귀)
- 회차 열기 기록(최근 본 회차 표시)
