# IA / 주요 화면 구조

## 사이트맵 (초안)
- 홈
- 탐색/검색
- 태그별 리스트
- 이야기 상세
- 읽기 화면
- 시리즈 페이지
- About/문의(선택)

## 핵심 사용자 흐름
1) 홈 진입 → 태그 필터 선택 → 리스트 탐색 → 상세 → 읽기
2) 검색 → 결과 리스트 → 상세 → 버전 전환 → 읽기
3) 시리즈 진입 → 순서대로 읽기

## 화면별 핵심 요소
### 홈
- 추천/신규/인기 섹션
- 빠른 필터(연령/길이)
- 이어보기(최근 N개)

### 리스트(탐색/태그)
- 카드: 제목, 연령, 길이, 테마
- 필터 바: 연령/길이/테마/분위기

### 이야기 상세
- 이야기 소개
- 버전 선택(연령/길이)
- 관련 이야기/시리즈

### 읽기 화면
- 본문 중심
- 글자 크기/줄 간격 조절
- 진행 표시(스크롤 %)
- 이어보기(마지막 읽은 위치로 복귀)
- 회차 열기 기록(최근 본 회차 표시)
