# 읽기 화면 스펙

## 1) 목적 / 핵심 행동
- 방해 없이 읽기에 집중하도록 제공
- 메타데이터(요약/태그/연령/길이/예상 시간)와 본문을 한 화면에서 제공
- 연작은 회차 단위로 읽기 가능

## 2) 주요 UI 구성요소
### 상단 영역
- 뒤로가기
- 제목/요약/태그/연령/길이/예상 시간
- 관련 버전 바로가기

### 본문 영역
- 큰 기본 글씨
- 충분한 줄 간격

### 설정 패널(필수)
- 글자 크기 조절
- 줄 간격 조절은 추후

### 연작 내 회차 이동
- 칩/목록으로 회차 바로가기
- 이전/다음 회차 버튼

## 3) 상태/예외
- 로딩: 스켈레톤 텍스트
- 회차 없음: “준비 중” 안내

## 4) 사용자 동작/전환
- 회차 열기 시점에 이어보기 기록 저장
- 회차 변경 → 본문 갱신

## 5) 데이터 요구사항
- version id, title, age_range, length_type
- 단편: 본문 텍스트
- 연작: chapters 배열(제목/본문/estimated_read_time)

## 6) 모바일 UX 규칙
- 스크롤은 본문에 집중
- 상단 UI는 스크롤 시 자동 숨김(선택)
