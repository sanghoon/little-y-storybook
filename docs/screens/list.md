# 리스트(탐색/검색) 화면 스펙

## 1) 목적 / 핵심 행동
- 태그/검색 결과를 빠르게 탐색하고 원하는 이야기를 선택

## 2) 주요 UI 구성요소
### 상단 영역
- 검색 입력(옵션)
- 현재 적용된 필터 표시(칩)

### 필터 바
- 연령, 길이, tags
- 정렬 옵션(신규/인기/추천) — 초기에는 숨김 가능

### 결과 리스트
- 카드 정보: 제목, 연령, 길이, 주요 태그
- 연작 여부 표시(연작 아이콘 또는 배지)

## 3) 상태/예외
- 로딩: 스켈레톤 리스트
- 결과 없음: “조건에 맞는 이야기가 없어요” 안내 + 필터 초기화 버튼

## 4) 사용자 동작/전환
- 카드 탭 → 읽기 화면(메타/본문 통합)
- 필터 변경 → 리스트 갱신
- 정렬 변경 → 리스트 재정렬

## 5) 데이터 요구사항
- version id, title, summary(선택), age_range, length_type, tags

## 6) 모바일 UX 규칙
- 필터 칩은 1줄 스크롤
- 카드 내 텍스트는 2줄 이상 넘지 않도록 말줄임
