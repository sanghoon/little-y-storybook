# 이야기 상세 화면 스펙

> 업데이트: 본 화면은 `/read/:id`에 통합되어 별도 상세 페이지를 노출하지 않습니다. 아래 스펙은 히스토리 참고용이며, 현재는 읽기 화면 상단에서 이야기 메타/버전 정보를 함께 제공합니다.

## 1) 목적 / 핵심 행동
- 이야기 소개와 버전 정보를 제공하고 읽기 시작을 유도

## 2) 주요 UI 구성요소
### 상단 영역
- 제목
- 요약
- 주요 태그

### 버전 선택 영역
- 버전 리스트(연령/길이 표시)
- 연작 표시
- 기본 선택 버전 강조

### 시작 버튼
- “읽기 시작” CTA

### 관련 콘텐츠(선택)
- 같은 태그의 다른 이야기

## 3) 상태/예외
- 버전 없음: “준비 중” 안내
- 콘텐츠 없음: 빈 상태 처리

## 4) 사용자 동작/전환
- 버전 선택 → CTA 대상 변경
- CTA 탭 → 읽기 화면(선택한 버전)

## 5) 데이터 요구사항
- story id, title, summary, tags
- versions: id, age_range, length_type, estimated_read_time, tags

## 6) 모바일 UX 규칙
- CTA는 화면 하단에 고정(선택)
- 버전 선택은 라디오/카드 형태
